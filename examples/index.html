Browser Example

This page loads the built ESM bundle directly from build/d3-cloud.js and renders an SVG word cloud with no bundler and no extra browser dependencies.

Serve the repo root with any static file server, then open /examples/. For raw text frequency analysis, see the frequency example. For image-based sprites, see the image sprite example. For a masked cloud, see the mask example. For custom font loading, see the font loading example. For background layout work, see the worker example.
